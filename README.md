# CSE 220 Homework 5 (Pointer Operations)
Name-Aninda Saprotiv Roy ID-116485388

This repository contains the implementation for Homework 5 (Spring 2026):

- Sparse matrix conversion
- Hadamard product
- Matrix multiplication
- Diagonal sum

All core logic is implemented in matrix.c using pointer dereferencing.

## Files

- [matrix.c](matrix.c): Required function implementations
- [matrix.h](matrix.h): Function declarations
- [test_matrix.c](test_matrix.c): Criterion unit tests
- [Makefile](Makefile): Build and test commands

## Build

```bash
make
```

## Run Tests

```bash
make test
```

## Clean
